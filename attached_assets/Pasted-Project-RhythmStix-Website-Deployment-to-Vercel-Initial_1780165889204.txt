## Current Theory

The frontend build succeeds.

Failure occurs during Vercel's TypeScript processing of the serverless API.

Vercel reports only:

api/index.ts: Emit skipped

but does not expose the underlying TypeScript error.

Likely causes:

1. A TypeScript error somewhere inside:
   artifacts/api-server/src/**

2. Vercel attempting to compile backend code as a serverless function and failing.

3. A missing dependency/type import used by one of the route modules.

4. A Vercel serverless function incompatibility.
